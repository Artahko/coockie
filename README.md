Головний файл app.py, у ньому запускається сайт. Інші допоміжні (мають функції, які використовуються в app.py)

Функція парсингу файлу знаходиться в parse.py, аналітика в analytics.py, візуалізація в visualization.py, a ШІ висновок в ai_summary.py


*Чому використані нами технології найкраще вирішують поставлені задачі?*

pandas дозволяє працювати з великими таблицями як з одним цілим, векторні операції по колонках.

pymap3d готова, перевірена реалізація перетворень координат з геодезичних (широта/довгота) у метричні (X, Y, Z) — geodetic2enu в один рядок замість ручної матричної алгебри.

plotly дозволяє створити не статичну картину, а інтерактивний 3D графік з анімацією через frames, colorbar, а також побачити швидкість через колір і отримати точні дані в кожній точці при наведенні курсору.

numpynp.stack для customdata в hover, векторне обчислення. Він забезпечує високу швидкість обробки даних і ми використали його для ефективного групування додаткової інформації (коорд, швидкість), яка відображається у підказках, щоб анімація не гальмувала.
